# Drop real photos here

Griffin's Tree Service's real job photos live on **Facebook** and **Yelp**, both of which
block automated download (FB token-gated, Yelp returns HTTP 403). So this concept ships with
tasteful on-brand placeholders. To make it real, drop the actual photos into this folder using
the exact filenames below, and they'll appear automatically (the HTML already points at them).

| Filename | Where it appears | Ideal shot |
|---|---|---|
| `hero.jpg` | Full-bleed hero background | A crane lift or a climber high in a big oak, action shot |
| `crane.jpg` | "Crane-assisted removals" feature row | The bucket/crane truck on a real job |
| `stump.jpg` | "Stump grinding" feature row | Stump grinder mid-cut, chips flying |
| `storm.jpg` | "Storm response" feature row | A downed tree on/near a house being cleared |
| `team.jpg` | "Why us" / since-2006 band | The crew in gear by the truck |
| `gallery-1.jpg` … `gallery-4.jpg` | Work gallery | Before/after removals, big fells, cleanup |

Recommended: landscape JPGs, ~1600px on the long edge, under ~400 KB each.

Source photos to pull from (owner has access):
- Facebook: https://www.facebook.com/griffinstreeservice/
- Yelp (24 photos): https://www.yelp.com/biz/griffins-tree-service-saint-amant-2
